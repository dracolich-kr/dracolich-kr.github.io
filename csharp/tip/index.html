    [Tip] C# 프로젝트 폴더가 다른 위치에 있는 cs파일 참조 By Global Using

    최대 1 분 소요

    목차

    기존 using 문의 경우 개별 파일마다 개별로 선언을 해서처리를 해야해서 매우 불편하고 사용하기 힘든 단점이 많은 기능이다.

    하지만 C# 10에 Global Using이 추가 되면서 이젠 프로젝트 단위로 하나의 cs 파일에 선언을 하면 다른 파일에서도 별도의 선언을 할 필요 없이 편하게 사용을 할 수 있게 되었다.

    주로 맴버 타입을 재정의 하는 용도로 나는 아래와 같이 사용하는데

    global using ActorIdx = Int32;
global using UserName = string;

    하지만 우리가 복수개의 프로젝트를 구성하는 경우 동일한 cs 파일을 매번 추가해야하는 단점이 있다.
    프로젝트 파일에 다음과 같이 파일을 추가를 해놓으면 그래도 덜 불편하게 동일한 C# 프로젝트에서 타입 설정을 편하게 설정 할 수 있다.

    <ItemGroup>
  <Compile Include="..\OtherProject\DefineName.cs">
    <Link>Shared\DefineName.cs</Link>
  </Compile>
</ItemGroup>
