    [MySql] MySql 이란?

    최대 1 분 소요

    목차

    • 개요
      • MyISAM
      • InnoDB
    • 참고 사이트
      • GUI 개발 툴

    개요

    1995년도에 발표된 RDBS(Relational Database System) 관계형 데이터 베이스이다.
    MySql은 스토리지 엔진에 따라 다양한 데이터 저장 방식을 지원
    대표적으로 MyISAM, InnoDB 가 존재 한다.

    MyISAM

    InnoDB가 나오기전에 사용되었던 기본 스토리지 엔진
    사용 방법이 간편하지만 트랜잭션이 불가능하고 데이터가 삭제가 되었을 때
    불필요한 데이터(OverHead)가 발생하여 현재는 InnoDB가 기본 스토리지 엔진으로 사용되고 있다.

    InnoDB

    MySql의 기본 스토리지 엔진이다.
    행 수준의 잠금 및 다중 사용자들의 동시성을 보장한다.
